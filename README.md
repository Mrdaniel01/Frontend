# NEWS: "EasyNews"

Diario de noticias varias de Argentina, Colombia, Venezuela y México.

## starting with 🚀

git clone git@github.com:Proyecto-Noticias/Frontend.git

### installation 🔧

npm i

## implementing the project 🔧

npm run start (Port: localhost:3000)
